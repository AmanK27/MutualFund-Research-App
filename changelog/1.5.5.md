# v1.5.5

## [1.5.5] - 2026-03-02

### Added
- **Manual Data Refresh Trigger:** Injected a "↻ Refresh" action button next to the main search bar to allow users to manually re-parse the live AMFI daily mutual fund data at-will, complete with loading animation feedback states.

### Fixed
- **JavaScript Regex Compilation Error:** Patched the `index.html` `categoryRegex` syntax where double backslashes (`\\s`) from a previous injection incorrectly evaluated as literal string slashes instead of whitespace matching groups inside the Native JavaScript RegExp prototype, which caused 100% of live categories to return 0 funds in production.
