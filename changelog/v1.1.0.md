# v1.1.0 — Firebase Auth & Firestore Integration

**Date:** 2026-03-02

## Overview
Added Firebase Authentication (Google Sign-In) and Firestore database integration. The app now requires sign-in before accessing the dashboard.

## Features Added

### Login Screen
- Premium glassmorphism overlay with animated background glow
- Google Sign-In button with official Google SVG icon
- Loading state and error handling during auth flow
- Privacy footer note

### Authentication Flow
- Firebase v9 compat SDK (Auth + Firestore) loaded via CDN
- `onAuthStateChanged` observer toggles login ↔ dashboard visibility
- Session persistence — stays signed in across browser refreshes
- Sign-out button in sidebar

### User Profile
- Google profile photo as avatar in sidebar
- Display name and email shown
- Auto-creates/updates user document in Firestore on each login

### Firestore Data Model
- Collection path: `users/{userId}/transactions/{transactionId}`
- Transaction fields: `type`, `schemeCode`, `schemeName`, `amount`, `units`, `navAtDate`, `date`, `createdAt`
- Helper functions ready for Phase 4: `addTransaction()`, `getTransactions()`, `deleteTransaction()`

## Technical Notes
- Uses Firebase v9 **compat** SDK to preserve global function scope in single-file architecture
- Firebase config set to project: `mutualfund-research-app`
- `#app` starts with `display: none` — shown only after auth
