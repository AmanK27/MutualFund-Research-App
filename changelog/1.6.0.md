# v1.6.0

## [1.6.0] - 2026-03-04

### Refactored
- **Modular Architecture:** Broke down the monolithic `index.html` file into a modular structure:
  - Extracted all styling to `css/styles.css`.
  - Extracted pure math and formatting functions to `js/utils.js`.
  - Extracted network fetch functions to `js/api.js`.
  - Extracted UI rendering and state management to `js/app.js`.

### Added
- **Guest / Development Mode:** Added a "Continue as Guest (Development Mode)" button to bypass Firebase authentication for local testing. It mocks transaction storage in local memory.
- **Holdings Retry Logic:** Added a manual "Retry" button for the Top 10 Holdings fetch in case the external API fails (mirroring the Category peers logic).

### Fixed
- **Refresh Button hijack:** Fixed a bug where clicking the "↻ Refresh" button on the main page mistakenly triggered a re-fetch and UI hijack to the Top Funds (Equity) panel instead of just updating the background AMFI data.
