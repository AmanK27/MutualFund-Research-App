# v1.2.0 — Math Engine & SIP Calculator

**Date:** 2026-03-02

## Overview
Implemented Phase 5 of the Mutual Fund Research Dashboard, focusing on advanced client-side mathematical calculations and a dynamic Historical SIP Calculator. All calculations run locally in the browser.

## Features Added

### Mathematical Engine
- **XIRR Utility:** Added a robust `calcXIRR` function using the Newton-Raphson approximation algorithm to calculate Internal Rate of Return for uneven cashflows.
- **Rolling Returns Math:** Added foundational logic (`calcRollingReturns`) to calculate average, minimum, and maximum returns over specific timeframes (preparatory for Phase 6).

### Interactive SIP Calculator UI
- Added a premium glassmorphism "SIP Calculator" card beneath the NAV chart.
- **Dynamic Inputs:** Users can specify Monthly Investment Amount (₹) and Duration (Years).
- **Historic Simulation:**
  - Calculates the exact dates SIP installments would have occurred historically.
  - Fetches the closest actual historical NAV for each installment date.
  - Computes the aggregate units purchased and prices them at the latest NAV.
- **Results Display:**
  - Shows total amount theoretically invested over the period.
  - Shows the final corpus value based on actual historical data.
  - Displays the Annualized XIRR calculated on the fly.
- **Real-time Recalculation:** The results instantaneously recalculate and update when input values are changed.

## Technical Notes
- The XIRR calculation limits iterations to 100 to prevent infinite loops in the browser for complex cashflow arrays.
- SIP generation handles edge-cases where the requested duration exceeds the fund's historical data availability by falling back to the fund's inception date.
