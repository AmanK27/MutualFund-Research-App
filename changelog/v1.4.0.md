# v1.4.0 - Personal Portfolio Tracker
- Added 'Add to Portfolio' dynamic modal allowing users to log mutual fund purchase transactions.
- Automated allocation calculations by fetching historical NAV from `mfapi.in` on exact transaction dates.
- Integrated Firestore writes to persist transaction records securely under the user's UID.
- Introduced a dedicated 'My Portfolio' glassmorphism sidebar section.
- Added data aggregation logic using Promise.all to fetch live NAVs, tabulate Current Value, and calculate Absolute Returns across all active holdings.
